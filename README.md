# 📉 Hybrid Drop-Off Investigation: B2B SaaS Onboarding

## 📌 The Business Problem
A simulated B2B SaaS platform was experiencing a high drop-off rate during the user onboarding phase. The goal of this mixed-methods project was to identify exactly *where* users were abandoning the setup process and discover *why* the cognitive friction was occurring.

## 🛠️ Methodology
1. **Quantitative (SQL & Pandas):** Extracted raw event logs to build a conversion funnel and calculate step-to-step drop-off rates.
2. **Qualitative (UXR):** Conducted a heuristic evaluation of the specific screen where the highest mathematical drop-off occurred.
